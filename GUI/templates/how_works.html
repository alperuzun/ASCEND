How ASCEND Works

Click on each step to learn more about ASCEND's workflow.

Step 1: Upload Your Data

Choose a CSV file containing gene expression data from the TCGA database.

Step 2: Select Cancer Type

Pick a cancer type (e.g., breast, lung) for biomarker identification.

Step 3: Choose Analysis Type

Select biomarker gene analysis, DNA methylation analysis, or both.

Step 4: Select Output Format

Choose CSV, visualization, or combined results.

Step 5: Data Processing

ASCEND analyzes the data and identifies significant biomarkers.

Step 6: Review Results

Explore detected biomarkers and methylation markers.
